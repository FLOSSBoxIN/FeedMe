FeedMe is an reader client application for RSS service:
- Feedly
- InoReader
- Bazqux
- The Older Reader
- Feedbin
- Tiny Tiny RSS
- FreshRSS
- Fever

This is the documentation of FeedMe.

## Version:
- release: 3.20.0
- beta: 3.20.4 (join beta test: https://play.google.com/apps/testing/com.seazon.feedme)
